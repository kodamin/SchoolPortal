About Student Portal

This single page application allows instructors to view their courses online and simplifies the management of courses. Anonymous users can view all courses. Instructors who are logged in can view their courses and edit them as they wish as well as add new courses. This site is intended to be intuitive, fast, and responsive. Courses can be sorted by category as well as searched. If there are too many courses the user can see the rest on separate pages.

Technologies:

  • HTML5
  • CSS3
  • Bootstrap
  • AngularJS
  • jQuery
